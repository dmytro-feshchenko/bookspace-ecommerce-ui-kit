# TextInput

The `TextInput` component represents complete form control.
It includes label, input, info message and place for error message. 

## Basic text input

```js live
<TextInput label="Standard input" variant="standard" />
```

### 

### Using props to customize the text

Modify the text to see it change live:
```js live
<Text text="hello from Text" />
```
